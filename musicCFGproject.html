About MusicCFG and How it Works

MusicCFG is Music for Code First Girls Project and the idea of this project is to combine Spotify and other Social Media platform's APIs to show user the top recommended Music choices and how many times was each song from the recommended list was mentioned on social media platforms. The aim is to analyse the popularity of the song in real-time. User can also listen to song by clicking on the audio button.
